# Chatwork_SpeedMention
Chatwork のメンション作成効率化ツール

## 機能
- 役職順に自動ソート
- 部署単位で名前リストを選択可能
	- リストのマウスホイール切り替え対応
- 名前あり・なしの切り替え
	- 名前あり：行単位でコピー
	- 名前なし：IDのみを横並びでコピー
